## Khi nao dung

- Them section moi cho page da co
- Tao feature component moi
- Sua primitive UI trong `src/Components/ui/`

## Inputs can doc

## Cach thuc hien

1. Tim component gan nhat da ton tai trong repo truoc khi tao moi.
2. Uu tien Tailwind utility classes cho thay doi moi.
3. Neu logic style chi phuc vu 1 page, dat tai feature component thay vi primitive UI.
4. Neu can sua primitive UI, giu API component on dinh de tranh gay vo dien rong.
5. Kiem tra responsive o breakpoint chinh neu thay doi layout.

## Verify

- Kiem tra visual state co loading, empty, error neu co

- Chay test lien quan neu component da co test
