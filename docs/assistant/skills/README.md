# Skills Library

## Muc dich

Thu muc nay luu cac skill card mo ta cach agent nen xu ly cac dang task lap lai trong FE.

## Skill khac instruction o dau

- `instructions/` tra loi "Repo nay la gi, quy tac chung la gi?"
- `skills/` tra loi "Gap dang bai nay thi xu ly theo cach nao?"

## Khi nao nen tao skill moi

- Cach lam do da duoc dung lap lai tu 2 lan tro len.
- Co checklist ro rang.
- Co the giup session sau bat dau nhanh hon.

## Format de xuat

Moi skill nen co:

- Muc dich
- Khi nao dung
- Inputs can doc
- Cac buoc thuc hien
- Verify
- Dau hieu can dung lai

## Ghi chu

Skill card nen ngan, de doc trong 1 den 2 phut.
