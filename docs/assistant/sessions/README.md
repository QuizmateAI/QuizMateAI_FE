# Session Summaries

## Muc dich

Thu muc nay luu lai moi session chat co thay doi dang ke de nhom co lich su ngan gon, de tra cuu, de tiep noi cong viec.

## Nguyen tac

- Moi session tao 1 file rieng.
- Moi file phai tu tong ket duoc ma khong can doc lai toan bo chat.
- Neu session khong co thay doi code nhung co thay doi ve instruction, van nen ghi lai.

## Dat ten file

- Dung format: `YYYY-MM-DD__chu-de-ngan.md`
- Vi du: `2026-04-02__payment-result-fix.md`

## Quy trinh cap nhat

1. Tao file moi tu `_session-template.md`
2. Dien nhanh cac muc bat buoc
3. Them dong moi vao `_index.md`

## Muc bat buoc trong moi session

- Request
- Scope
- Files changed
- Summary of changes
- Verification
- Risks or follow-ups
